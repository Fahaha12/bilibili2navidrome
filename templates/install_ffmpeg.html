FFmpeg安装指南

为什么需要安装FFmpeg？

FFmpeg是一个用于处理多媒体文件的开源工具集，本应用需要它来将下载的音频转换为MP3格式。 请按照您的操作系统选择下面的安装方法。

  1. 访问 FFmpeg官网
  2. 在"Get packages & executable files"部分，点击"Windows builds from gyan.dev"
  3. 下载最新版本（文件名如：ffmpeg-release-full.7z）
  4. 解压下载的文件到任意目录（如：C:\ffmpeg）
  5. 将FFmpeg添加到系统PATH：
    • 按下Win+R，输入sysdm.cpl并回车
    • 转到"高级"选项卡，点击"环境变量"
    • 在"系统变量"部分，找到并选择Path变量
    • 点击"编辑"，然后"新建"
    • 添加FFmpeg的bin目录路径（如：C:\ffmpeg\bin）
    • 点击"确定"保存所有更改
  6. 验证安装：
    1. 打开命令提示符（CMD）
    2. 输入：ffmpeg -version
    3. 如果看到版本信息，说明安装成功

  1. 打开终端（Terminal）
  2. 安装Homebrew（如果尚未安装）： 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  3. 通过Homebrew安装FFmpeg： 
    brew install ffmpeg
  4. 验证安装： 
    ffmpeg -version

  1. 打开终端
  2. 对于基于Debian的系统（如Ubuntu）： 
    sudo apt update
sudo apt install ffmpeg
  3. 对于基于RPM的系统（如Fedora）： 
    sudo dnf install ffmpeg
  4. 验证安装： 
    ffmpeg -version
